# Talks, Articles & Press

If you want to see vCon explained by the people building it, here are the canonical sources. We pull from IETF working-group sessions, TADSummit and TADHack keynotes, podcasts (Jeff Pulver's Substack and others), and longform writing across Medium, Telecom Reseller, and the vendor blogs that have picked vCon up.

Use this section to:

* **Get oriented quickly.** Pick one talk and one article from the top of each subsection.
* **Cite vCon.** When you need a durable link to a vCon explanation for a slide deck, blog post, or RFP, prefer the items pinned at the top of each list.
* **Track the working group.** The [IETF Sessions](ietf-sessions.md) page is the closest thing to a primary source for where the spec is going.

## Subsections

* [IETF Sessions](ietf-sessions.md) — every IETF VCON working-group session with recording and minutes
* [Conference Keynotes](conference-keynotes.md) — TADSummit, TADHack, Generative AI Expo, and other event talks
* [Podcasts](podcasts.md) — audio interviews, including Jeff Pulver's ongoing Substack series
* [Articles & Press](articles-and-press.md) — longform writing, press coverage, and LinkedIn commentary

## Submitting links

If you've recorded a talk, written an article, or run a podcast episode about vCon and you want it added, the lightest path is a pull request against `talks-articles-press/` in [vcon-dev/vcon-docs](https://github.com/vcon-dev/vcon-docs). Keep entries to one line plus a short takeaway; reverse-chronological order within each subsection.
